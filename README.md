## Description
Basic authentication system using nestJS.

## Installation

```bash
$ npm install

# or, using yarn
$ yarn install
```

## Running the app

```bash
# development
$ npm run start  # yarn start

# watch mode
$ npm run dev  # yarn dev

# production mode
$ npm run prod  # yarn prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:coverage
```
